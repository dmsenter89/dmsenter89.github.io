I love simple CLI tools and am a big fan of the [Unix philosophy](https://en.wikipedia.org/wiki/Unix_philosophy). 
Recently I came across [The Dam](https://the-dam.org/), a public Unix server that implements a clever tool
they termed suc - the Simple Unix Chat. Essentially, it applies the Unix philophy to create a simple chat tool
that can be used on any modern Unix server. The key code consists of just a few lines of Bash code.
Check out the documentation for it [here](https://the-dam.org/docs/explanations/suc.html).
