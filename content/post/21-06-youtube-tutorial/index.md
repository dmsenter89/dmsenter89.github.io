I have been using both SAS and Python extensively for a while now. With each having great features, it was very useful to combine my 
skills in both languages by seamlessly moving between SAS and Python in
a single notebook. In the video below, fellow SAS intern Ariel Chien and I show how easy it is to connect the SAS and Python kernels using the open-source SASPy package together with SAS OnDemand for Academics.
I hope you will also find that this adds to your workflow!


{{< youtube 6mcsbeKwSqM >}}


The Jupyter notebook from the video can be viewed [on GitHub](https://github.com/sascommunities/sas-howto-tutorials/tree/master/sastopython). For installation instructions, check out the [SASPy GitHub page](https://github.com/sassoftware/saspy). Configuration for SASPy to connect to ODA can be found [at this support page](https://support.sas.com/ondemand/saspy.html). For more information on SAS OnDemand for Academics, [click here](https://www.sas.com/en_us/software/on-demand-for-academics.html).
